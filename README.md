# PluralCloud

É uma uma solução de cloud privada para uso em Labs afim de explorar ferramentas como:

- Terraform
- Ansible
- Kubernetes
- Docker
- Vagrant

## Setup

```shell
git clone -b develop https://github.com/angolarti/pluralsiscloud.git
```

```shell
cd pluralsiscloud
```

```shell
go mod tidy
```

```shell
npm run dev
```
